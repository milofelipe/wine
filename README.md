# Wine API

### Uses

* Java 8
* Maven

### Build

`$ ./mvnw package `

### Run Locally

`$ ./mvnw spring-boot:run`

Then access Swagger UI to test available REST APIs:

[http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)

Mock database only contains three lot codes:
* 11YVCHAR001
* 11YVCHAR002
* 15MPPN002-VK
